* Overview

This document describes the org-agenda setup in Doom Emacs. The agenda uses
org-roam as its single source of truth — all machines share the same agenda
view, with work and personal contexts distinguished by tags, not file location.

*Key packages:*
- =org-super-agenda= — groups items under styled section headers
- =org-modern= — badge-style TODO keywords and clean tag labels
- =nerd-icons= — category icons for visual scanning
- =olivetti= — centered layout for focused viewing

* Agenda Views

** Overview Command (=o=)

The primary agenda view. Access via =SPC o a o=.

*Block 1 — Day Agenda:*
Shows today's scheduled items and deadlines, grouped into:

| Group   | Contents                        | Color   |
|---------+---------------------------------+---------|
| Overdue | Past-due scheduled and deadlines | Red     |
| Today   | Today's time grid and items     | Yellow  |

*Block 2 — All TODOs:*

| Group    | Filter        | Color  |
|----------+---------------+--------|

| Meetings | =@meeting= tag  | Purple |
| Reviews  | =@review= tag   | Blue   |
| Projects | =@project= tag  | Purple |
| Admin    | =@admin= tag    | Orange |
| Reports  | =@report= tag   | Grey   |
| Home     | =@home= tag     | Aqua   |

| Finance  | =@finance= tag  | Yellow |

* Workflow

** Daily Capture and Review

1. *Capture tasks:* =SPC X= then select template key (=t= for task, =m= for meeting, etc.)
2. *Review agenda:* =SPC o a o= to open the Overview
3. *Act on items:* =t= to cycle state, =s= to schedule, =,= to set priority
4. *End-of-day refile:* =SPC r B= to batch refile tagged tasks to area notes

** Area Notes

Tasks tagged with personal context tags (@home, @errands, etc.) can be
refiled to area notes for longer-term tracking:

| Tag        | Area File    |
|------------+--------------|
| =@home=      | home.org     |
| =@errands=   | errands.org  |
| =@online=    | online.org   |
| =@health=    | health.org   |
| =@finance=   | finance.org  |
| =@someday=   | someday.org  |
| =@learning=  | learning.org |

** Refile Keybindings

| Keybinding | Command                   | Description                      |
|------------+---------------------------+----------------------------------|
| =SPC r R=    | hd/refile-to-area         | Smart refile based on tag        |
| =SPC r B=    | hd/refile-all-daily-tasks | Batch refile all tagged tasks    |
| =SPC r a=    | hd/goto-area              | Jump to an area note             |

* Customization

** Adding a New Tag

1. Add the tag to =org-tag-alist= in =config.org= (pick an unused key)
2. Add a face to =org-tag-faces= with an Everforest color
3. Add a super-agenda group in the Overview command's =alltodo= block
4. Add a capture template if quick-capture is wanted
5. Optionally add a category icon mapping
6. Update this document

** Adding a New Super-Agenda Group

Add an entry to the =org-super-agenda-groups= list in the Overview command.
Groups are matched in order — first match wins. Use =:order= to control
display position.
