\section{Overview}
\label{sec:org6552951}

This document contains keybindings for daily use of Org-mode in Doom Emacs.
All keybindings use \texttt{SPC} as the leader key (Evil mode).

\textbf{Legend:}
\begin{itemize}
\item \texttt{SPC} - Space (leader key)
\item \texttt{C-} - Control
\item \texttt{M-} - Meta/Alt
\item \texttt{S-} - Shift
\item \texttt{RET} - Enter/Return
\end{itemize}

\section{Exporting This Document}
\label{sec:org7025705}

To export this document as PDF:

\begin{enumerate}
\item Open this file in Emacs
\item Press \texttt{SPC m e} (or \texttt{C-c C-e})
\item Press \texttt{l p} for \LaTeX{} to PDF
\item Or press \texttt{l o} to generate PDF and open it
\end{enumerate}

Requirements:
\begin{itemize}
\item \LaTeX{} distribution (\TeX{} Live, MacTeX, or MiKTeX)
\item \texttt{pdflatex} in your PATH
\end{itemize}
