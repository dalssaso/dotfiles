* Overview

This document contains keybindings for daily use of Org-mode in Doom Emacs.
All keybindings use =SPC= as the leader key (Evil mode).

*Legend:*
- =SPC= - Space (leader key)
- =C-= - Control
- =M-= - Meta/Alt
- =S-= - Shift
- =RET= - Enter/Return

* Area Notes (Personal Only)

Personal task management uses org-roam area notes. Tasks are captured to dailies,
then refiled to area notes based on tags. Completed tasks return to the daily as a log.

** Area Notes

| Area     | File         | Tag        |
|----------+--------------+------------|
| Home     | home.org     | =@home=      |
| Errands  | errands.org  | =@errands=   |
| Online   | online.org   | =@online=    |
| Health   | health.org   | =@health=    |
| Finance  | finance.org  | =@finance=   |
| Someday  | someday.org  | =@someday=   |
| Learning | learning.org | =@learning=  |

** Area Keybindings

| Keybinding | Command                 | Description                          |
|------------+-------------------------+--------------------------------------|
| =SPC r R=    | hd/refile-to-area       | Smart refile based on task's tag     |
| =SPC r B=    | hd/refile-all-daily-tasks | Batch refile all tagged tasks (review) |
| =SPC r a=    | hd/goto-area            | Jump to an area note (menu)          |

** Daily-First Workflow

1. =SPC r c t= - Capture task to today's daily
2. =SPC m q= - Add context tag (@home, @errands, etc.)
3. =SPC r R= - Smart refile single task, OR =SPC r B= - Batch refile all tagged tasks
4. =t= - Mark DONE in area note â†’ task moves to daily's Completed section

** End-of-Day Review

Run =SPC r B= to refile all tagged TODO tasks from today's daily to their
respective area notes at once. This fits a GTD-style review workflow.

* Exporting This Document

To export this document as PDF:

1. Open this file in Emacs
2. Press =SPC m e= (or =C-c C-e=)
3. Press =l p= for LaTeX to PDF
4. Or press =l o= to generate PDF and open it

Requirements:
- LaTeX distribution (TeX Live, MacTeX, or MiKTeX)
- =pdflatex= in your PATH
